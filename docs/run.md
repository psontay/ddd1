## This project use **Docker Compose** to run a Spring Boot application with **PostgreSQL** and **Redis**

```env

POSTGRES_CONTAINER_NAME=
POSTGRES_NETWORKS=

REDIS_CONTAINER_NAME=
REDIS_NETWORKS=
REDIS_PORT=":6379"
